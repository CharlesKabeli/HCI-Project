BUKANTSWE: Electronic Sesotho-English dictionary

Bukantswe Online - Go directly to online dictionary (now with 10 072 terms in Sesotho with English equivalents)


BUKANTSWE is a downloadable electronic Sesotho-English dictionary available only from this site. The dictionary consists of over 8000 (English and Sesotho combined) entries.

Here are some examples of screen shots in Bukantswe 1.2:

Searches can be done in English and Sesotho with the option of just looking for a single "whole word" or any instance of the word.

A new addition to version 1.2 is the word analysis function with which words can be broken up in terms of the root verb furthermore the possible class of nouns be determined. At this stage this function is fairly basic and might be improved on later.

Some spelling and translation mistakes found in version 1.3 of the dictionary file have been corrected.
Thank you for Thabo Manong in this regard.
